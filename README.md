# GeneracionDestruccionTerreno_TFG
Trabajo de fin de grado Videojuegos

El objetivo del proyecto es la generación mediante el uso de las mallas de vertices en Unity de un mundo de gran tamaño que se encuentre cargado en todo momento que permita la deformación de la malla que lo compone según unos valores de resistencia.

Ahora mismo el proyecto se compone de:

- Generación de mapa de valores del mundo según parámetros establecidos por el usuario
- Uso de este mapa para el renderizado del terreno mediante un sistema de carga de chunks de forma dinámica (En proceso)
- Deformación del terreno según potencia y tamaño del impacto (En proceso)
